Which Chart for Which Data?

A one-page decision guide — start from what you want to say, not the chart you like.   Availability: Excel native in Excel   Adv Plotly / specialist tools

Ask first: What's the message? (compare · trend · part-of-whole · spread · link · flow) How many variables? Categorical or numeric? How many data points?

📈 Change over time

"How has this moved across time?"
  • Line chart — 1+ series over time Excel
  • Area / stacked area — magnitude + composition Excel
  • Column chart — few, discrete periods Excel
  • Slope chart — just two time points Adv
  • Candlestick / OHLC — price data Excel
  • Index chart — rebase series to 100 Adv

📊 Comparison

"Which is bigger / ranks higher?"
  • Bar chart — ranking, long labels Excel
  • Column chart — few categories Excel
  • Grouped bar/column — 2 categories Excel
  • Dot plot — many items, tidy Adv
  • Dumbbell / range — two values each Adv
  • Radar — one item on many metrics Excel

🥧 Part-to-whole

"What share of the total?"
  • Stacked bar/column — shares over items/time Excel
  • 100% stacked — proportions only Excel
  • Pie / doughnut — 2–4 slices max Excel
  • Treemap — many nested parts Excel
  • Waffle chart — a single percentage Adv
  • Waterfall — how a total is built up Excel

🔔 Distribution / spread

"How are values spread out?"
  • Histogram — shape of one variable Excel
  • Box & whisker — compare spreads Excel
  • Violin plot — spread + density Adv
  • Strip / beeswarm — every point shown Adv
  • Density / ECDF — smooth or cumulative Adv
  • Pareto — ranked freq + cumulative Excel

🔗 Relationship

"Do two things move together?"
  • Scatter plot — 2 numeric variables Excel
  • Bubble chart — add size = 3rd var Excel
  • Connected scatter — pair + time path Adv
  • Heatmap — correlation matrix Adv
  • 2D density / hexbin — many points Adv
  • Trend line + scatter — show the fit Excel

🧊 Many dimensions (3+)

"Lots of variables at once?"
  • Bubble chart — x, y, size, colour (4D) Excel
  • 3D scatter / bubble — up to 5D Adv
  • Parallel coordinates — 5–15 vars Adv
  • Scatterplot matrix (SPLOM) — all pairs Adv
  • Small multiples — facet by a variable Adv
  • PCA / t-SNE / UMAP scatter — compress Adv

🔀 Flow & process

"How does volume move between stages?"
  • Sankey diagram — flows between nodes Adv
  • Funnel — drop-off through stages Excel
  • Alluvial / parallel categories Adv
  • Streamgraph — flows over time Adv

🗺️ Geospatial

"Where is it happening?"
  • Choropleth (filled map) — value by region Excel
  • Bubble / symbol map — size at points Adv
  • Cartogram — resize by value Adv
  • Density / flow map — hotspots, routes Adv

🌳 Hierarchy

"Nested categories / structure?"
  • Treemap — size = value, nested Excel
  • Sunburst — radial hierarchy Excel
  • Icicle — rectangular hierarchy Adv
  • Dendrogram / tree — branching links Adv

Rules of thumb

  • One message per chart. If you need a paragraph to explain it, split it in two.
  • Bar beats pie whenever you have more than ~4 categories — length is easier to judge than angle.
  • Start bar/column axes at zero; line charts for trends need not.
  • Up to ~4 numeric dimensions → bubble or 3D bubble. Beyond that → parallel coordinates, SPLOM, or a dimensionality-reduction scatter.
  • Order categories by value, not alphabetically, unless there's a natural sequence.
  • Label directly instead of relying on a legend where you can.

Avoid

  • 3D bar/pie for effect — it distorts the values.
  • Dual y-axes that imply a relationship that isn't there.
  • Too many colours; reserve colour for meaning.
  • Truncated axes that exaggerate small differences.
